## **🎯 你需要掌握的核心技能**

### **1. AI测试思维（最重要）**
```python
# 传统测试思维：
# 输入A → 期望输出B → 验证是否=B

# AI测试思维：
# 输入A → 可能输出B/C/D → 评估概率和准确性
# 关注：准确率、延迟、稳定性、公平性

# 需要理解的AI测试特点：
# 1. 结果不确定性 - 同样输入可能不同输出
# 2. 性能要求高 - AI计算通常很耗时
# 3. 质量指标多 - 不只是对错，还有准确率、召回率等
# 4. 系统复杂 - 涉及模型、推理引擎、硬件等
```

### **2. AI模型基础知识**
```python
# 不需要会训练模型，但需要理解：
# 1. 模型类型：分类、回归、生成模型
# 2. 模型输入输出：特征向量、预测概率
# 3. 模型评估指标：准确率、精确率、召回率、F1
# 4. 模型性能：延迟、吞吐量、内存使用

# 实用理解：
# 分类模型 → 判断类别（猫/狗、正常/异常）
# 回归模型 → 预测数值（房价、温度）
# 生成模型 → 创造内容（文本、图像）
```

### **3. 推理引擎技术（岗位核心）**
```python
# 这是岗位要求的核心技能：
# 1. vLLM - 高吞吐量推理引擎

# 你需要理解：
# 为什么需要这些优化？
# 它们解决了什么问题？
# 如何测试这些优化效果？
```

### **4. 性能测试技能**
```python
# 传统性能测试 → AI性能测试

# 并发数 → 吞吐量（tokens/s）

# 1. 延迟：单次推理时间
# 2. 吞吐量：每秒处理请求数
# 3. 资源使用：GPU利用率、显存占用
# 4. 并发性能：多用户同时使用时的表现
```
